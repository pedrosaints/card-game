# card-game
App de Card Game destinado a Disciplina de Engenharia de Software - UFC 2021.1
Versão Unity 2020.3.12f1
# Como executar:
  Abra a Unity, vá em assets/AllScenes, substitua a cena do editor para StartGameScene (arraste a cena StartGameScene para a aba Hierarchy da Unity, depois remova a cena que estava no editor, clicando com o botão direito do mouse sobre a cena e clicando em Remove Scene), agora, execute o projeto.
